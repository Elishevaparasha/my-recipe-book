# 🚀 הוראות העלאה לאינטרנט - שלב אחר שלב

## אופציה 1: Netlify (הכי קל! מומלץ למתחילים) ⭐

### שלב 1: הכנה
1. ודא שכל הקבצים בתיקייה אחת
2. התיקייה צריכה להכיל:
   - index.html
   - recipes.html
   - recipe-view.html
   - settings.html
   - manifest.json
   - תיקיית css/
   - תיקיית js/
   - תיקיית data/

### שלב 2: הרשמה ל-Netlify
1. היכנס ל: https://www.netlify.com
2. לחץ על "Sign up" (הרשמה)
3. בחר "Sign up with Email" או התחבר עם GitHub/Google
4. מלא את הפרטים והירשם (חינם לגמרי!)

### שלב 3: העלאת האתר
1. לאחר ההתחברות, תגיע לדף הבית של Netlify
2. לחץ על "Add new site" (הוסף אתר חדש)
3. בחר "Deploy manually" (העלאה ידנית)
4. **גרור את כל התיקייה** לאזור המסומן
5. המתן כ-30 שניות...
6. 🎉 **האתר שלך באוויר!**

### שלב 4: שינוי שם האתר (אופציונלי)
1. לחץ על "Site settings"
2. לחץ על "Change site name"
3. בחר שם (לדוגמה: `elisheva-recipes`)
4. הקישור שלך יהיה: `https://elisheva-recipes.netlify.app`

### שלב 5: שיתוף
- העתק את הקישור ושלח למורה! 🎓

## אופציה 2: GitHub Pages (למי שיש GitHub)

### שלב 1: יצירת Repository
1. היכנס ל: https://github.com
2. לחץ על "New repository" (ירוק)
3. שם ל-Repository: `recipe-book`
4. בחר "Public"
5. לחץ "Create repository"

### שלב 2: העלאת קבצים
1. לחץ על "uploading an existing file"
2. גרור את **כל הקבצים והתיקיות**
3. כתוב הודעה: "Initial commit"
4. לחץ "Commit changes"

### שלב 3: הפעלת GitHub Pages
1. לחץ על "Settings" (למעלה)
2. גלול למטה ל-"Pages" (בצד שמאל)
3. תחת "Source" בחר "main" branch
4. לחץ "Save"
5. המתן דקה...
6. הקישור יופיע למעלה: `https://[username].github.io/recipe-book`

## אופציה 3: Vercel (מהיר ומקצועי)

### שלב 1: הרשמה
1. היכנס ל: https://vercel.com
2. לחץ "Sign Up"
3. התחבר עם GitHub/Google/Email

### שלב 2: העלאה
1. לחץ "Add New..." > "Project"
2. לחץ "Browse" או גרור את התיקייה
3. לחץ "Deploy"
4. המתן כ-30 שניות
5. 🎉 האתר באוויר!

### שלב 3: קישור
- הקישור יהיה משהו כמו: `https://recipe-book-xxx.vercel.app`

### שלב 1: הרשמה

3. אם אין - בחר "Deploy from Git"

### שלב 3: העלאה

2. האתר יעלה אוטומטית
3. תקבל קישור
