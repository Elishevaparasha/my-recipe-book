# 🍳 ספר המתכונים המטורף - גרסה מתקדמת

אתר מתכונים אינטראקטיבי מתקדם עם פונקציות מטורפות שיפילו את כולם! 🚀

## ✨ תכונות מטורפות

### 🎨 עיצוב ואנימציות
- **6 ערכות צבעים מדהימות**: סגול קסום, כחול אוקיינוס, ירוק טבעי, ורוד שקיעה, כהה מסתורי, בהיר נקי
- **אפקט זכוכית (Glass Morphism)**: כרטיסים עם אפקט blur מטורף
- **אנימציות חלקות**: Fade-in, Pulse, Bounce, Shimmer
- **רקע מונפש**: גרדיאנט שמשתנה בזמן אמת
- **מעברים חלקים**: כל אלמנט עם אנימציית מעבר

### 🖼️ תמונות ומדיה
- תמונות למתכונים (מ-Unsplash)
- תמונה ראשית בדף המתכון
- תמונות ברשימת המתכונים
- תמיכה בהעלאת קישורים לתמונות

### ⭐ דירוגים ומועדפים
- מערכת דירוג 5 כוכבים
- חישוב דירוג ממוצע
- הוספה למועדפים עם אנימציה
- סינון לפי מועדפים
- שמירת דירוג אישי לכל משתמש

### 🏷️ תגיות וסינון מתקדם
- תגיות למתכונים (קל, מהיר, בריא, וכו')
- סינון לפי קטגוריה
- סינון לפי תגיות
- חיפוש מתקדם בשם, קטגוריה ותגיות

### 🔊 הקראה קולית משודרגת
- הקראה קולית בעברית
- כפתורי בקרה: התחל, השהה, המשך, עצור
- הדגשת השורה הנוכחית
- התאמת מהירות הקראה (0.5x - 2.0x)
- זמן המתנה מותאם בין שורות
- **צלילים**: ביפ בתחילה, סוף ובין שלבים

### 📊 סטטיסטיקות
- מספר מתכונים במערכת
- מספר מועדפים
- מתכונים שנצפו
- מתכונים שהוכנו
- זמן בישול כולל
- צפיות למתכון
- מספר דירוגים

### 📱 PWA - Progressive Web App
- ניתן להתקנה כאפליקציה
- עובד גם אופליין (לאחר טעינה ראשונה)
- אייקון באפליקציה
- מסך מלא ללא דפדפן

### 🎯 תכונות נוספות
- **הדפסה**: הדפסת מתכון עם עיצוב מותאם
- **רספונסיבי**: עובד מעולה במובייל, טאבלט ומחשב
- **נגישות**: תמיכה בקוראי מסך
- **טיפים**: אפשרות להוסיף טיפים למתכון
- **מידע מפורט**: זמן הכנה, בישול, מספר מנות, רמת קושי
- **התראות**: הודעות על פעולות שבוצעו
- **אנימציית קונפטי**: בשמירת הגדרות

## 🚀 התחלה מהירה

### הרצה מקומית

1. פתח את התיקייה בדפדפן
2. פתח את הקובץ `index.html`
3. התחבר עם אחד מהמשתמשים:
   - `elisheva` / `1234`
   - `admin` / `admin123`

### העלאה לאינטרנט (חינם)

#### אופציה 1: Netlify (מומלץ!)

1. היכנס ל-https://www.netlify.com
2. הירשם (חינם)
3. לחץ על "Add new site" > "Deploy manually"
4. גרור את **כל התיקייה** לאזור ההעלאה
5. תקבל קישור מיידי! 🎉
6. אפשר לשנות את שם הקישור בהגדרות

#### אופציה 2: GitHub Pages

1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים
3. עבור ל-Settings > Pages

> 🔶 ברירת המחדל של האתר עכשיו היא ערכת "חום אלגנטי" – הגדרות מאפשרות לבחור ערכת צבע רקע מתקדמת.
4. בחר Branch: main ולחץ Save
5. האתר יהיה זמין ב: `https://[username].github.io/[repo-name]`

#### אופציה 3: Vercel

1. היכנס ל-https://vercel.com
2. לחץ על "New Project"
3. העלה את התיקייה
4. האתר יעלה אוטומטית

## 📁 מבנה הפרויקט

```
project/
├── index.html              # דף התחברות מעוצב
├── recipes.html            # רשימת מתכונים עם סינון
├── recipe-view.html        # תצוגת מתכון + הקראה
├── settings.html           # הגדרות מתקדמות (כולל בחירת צבע רקע, ברירת מחדל: חום אלגנטי)
├── manifest.json           # PWA manifest
├── css/
│   └── style.css          # עיצוב מטורף עם אנימציות
├── js/
│   ├── auth.js            # ניהול התחברות
│   ├── storage.js         # ניהול localStorage מתקדם
│   ├── speech.js          # הקראה קולית עם צלילים
└── data/
    └── initial-data.js    # 6 מתכונים מלאים
```

## 🎯 שימוש באתר

### התחברות
- הזן שם משתמש וסיסמה
- לחץ על "כניסה למערכת"
- אנימציה מגניבה במעבר

### רשימת מתכונים
- **חיפוש**: חפש לפי שם, קטגוריה או תגית
- **סינון**: לחץ על כפתורי הסינון (הכל, מועדפים, קטגוריות)
- **מועדפים**: לחץ על הלב להוספה/הסרה
- **צפייה**: לחץ על כרטיס המתכון
- **הוספה**: כפתור "הוספת מתכון חדש"

### תצוגת מתכון
- **תמונה**: תמונה גדולה של המתכון
- **מידע**: זמן הכנה, בישול, מנות, קושי
- **דירוג**: לחץ על הכוכבים לדירוג
- **מועדפים**: הוסף/הסר ממועדפים
- **הקראה**: כפתורי בקרה מלאים
- **הדפסה**: הדפס את המתכון
- **סטטיסטיקות**: צפיות, דירוגים, זמן

### הגדרות
- **ערכת צבעים**: 6 אפשרויות מדהימות
- **גודל גופן**: קטן/בינוני/גדול
- **זמן המתנה**: 0.5-10 שניות
- **מהירות הקראה**: 0.5x-2.0x
- **אפקטי צליל**: הפעל/כבה
- **התראות**: הפעל/כבה
- **בדיקת הקראה**: בדוק לפני שמירה
- **איפוס**: חזרה להגדרות ברירת מחדל
- **סטטיסטיקות**: מתכונים שנצפו, הוכנו, זמן בישול
